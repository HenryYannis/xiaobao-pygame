## 🎨 核心编写准则 (五大法则)

为确保教学的零起点友好性与代码的高可读性，本仓库所有课程代码严格遵循以下 **五大核心法则**：

1. **变量命名全拼音化**：所有变量名、函数名使用全拼音（例如 `feichuan_sudu = 5`），消除初学者对英语词汇的恐惧感，让代码逻辑一目了然。
2. **“三位一体”逐行注释**：**每一行**非空代码均配有详尽的中文注释，注释包含“是什么（功能）+ 为什么（原理）+ 怎么用（关键参数说明）”。
3. **严格的“40行”与“3-5课”限制**：单节课的核心代码量（不含注释和空行）控制在 **30-40行** 之间；每个小游戏在 **3-5节课** 内完成，帮助学生快速获得完整成就感，杜绝疲劳。
4. **零外部素材原则**：所有界面绘制均使用 `pygame.draw`（圆形、矩形等）完成，不使用任何外部图片和音效。避免因路径问题导致报错，让学生百分百专注于算法与逻辑本身。
5. **增量标注规范**：新课相较于前一课新增的代码段上方，均明确标注 `# 新增变量：`、`# 新增函数：` 或 `# 新增代码：`，方便追踪进度。

## 🚀 快速开始

### 1. 运行环境配置
确保您的电脑上已安装 **Python 3.12 或更高版本**。
然后，打开终端并运行以下命令，一键安装唯一的第三方库依赖 **Pygame**：

```bash
pip install -r requirements.txt
```

### 2. 交互式游戏图形运行菜单 (强烈推荐)
为了方便直观地浏览和启动各课时游戏，我们使用 **Pygame** 开发了一个精美的 **GUI 启动器**。在仓库根目录下运行以下命令即可启动：

```bash
python run_launcher.py
```

启动器特色：
* **精美暗黑主题**：支持平滑的鼠标悬停高亮、发光边框与选项选中状态。
* **分栏布局**：左侧为 18 个游戏列表，右侧为当前选中的课时卡片。卡片支持友好解析（如 `Lesson 1 - 气泡浮现`）和真实文件名对照展示。
* **快捷链接**：左下角常驻作者和网址，点击可自动通过浏览器跳转至官网。
